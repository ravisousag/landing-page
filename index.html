Bem-vindo ao Tutor IA de Python Online!

Aqui você encontrará uma variedade de recursos para aprender Python de forma interativa e eficiente.

Sobre o Projeto

Este projeto visa criar um tutor inteligente para auxiliar no aprendizado da linguagem de programação Python.

Objetivos

Os principais objetivos do projeto são:

  • Fornecer um ambiente interativo de aprendizado.
  • Utilizar técnicas de inteligência artificial para personalizar o ensino.
  • Criar uma página web para aprender Python com IA

Disciplina

Este projeto foi desenvolvido para a disciplina INE5646 - Programação para WEB.
